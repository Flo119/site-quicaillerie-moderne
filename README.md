# Gestion Quincaillerie Moderne

Ce projet est une application web de gestion de quincaillerie créée avec PHP, HTML, CSS (Bootstrap 5) et JavaScript.

## Prérequis
- PHP installé sur votre machine.
- Un navigateur web moderne.

## Installation & Lancement

1. **Base de données** : Le fichier `database.sqlite` sera créé automatiquement lors du premier lancement.
2. **Utilisateurs par défaut** :
   - **Administrateur** :
     - Email : `admin@quincaillerie.com`
     - Mot de passe : `admin123`

## Comment lancer le site ?

### Option 1 : Via la ligne de commande (Recommandé)
Ouvrez votre terminal dans ce dossier (`GestionQuincaillerie`) et lancez la commande suivante :

```bash
php -S localhost:8000
```

Ensuite, ouvrez votre navigateur et allez sur : [http://localhost:8000](http://localhost:8000)

### Option 2 : Avec XAMPP / WAMP
1. Copiez tout le dossier `GestionQuincaillerie` dans votre dossier `htdocs` (XAMPP) ou `www` (WAMP).
2. Accédez-y via `http://localhost/GestionQuincaillerie`.

## Fonctionnalités
- **Accueil Moderne** : Design soigné avec effets de verre (Glassmorphism), animations SVG et transitions douces.
- **Catalogue** : Affichage des produits avec prix en XFA et images calibrées.
- **Espace Admin** : Ajout, suppression et modification (simplifiée) des produits.
- **Compte Client** : Inscription et connexion.
- **Chat** : Widget flottant avec liens directs vers WhatsApp/Telegram.
